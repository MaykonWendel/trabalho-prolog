# Projeto em Prolog

Este repositório contém um programa desenvolvido em **Prolog**, utilizando fatos e regras para representar informações e realizar inferências lógicas.

O código faz uso de programação lógica declarativa, permitindo a definição de relações e a realização de consultas a partir dessas relações.

## Conteúdo

- Arquivo `.pl` com fatos e regras em Prolog
- Predicados definidos para representar relações entre entidades
- Uso de recursão e unificação para responder consultas

##

- O projeto foi desenvolvido com fins acadêmicos
- O código pode ser executado em qualquer ambiente compatível com Prolog.
